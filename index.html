Ankit Sanjyal

I am an M.S. Data Science researcher at Fordham University, where I work as a Graduate Research Assistant in the Educational Data Mining Lab and as a Teaching Assistant for graduate Machine Learning courses.

My research focuses on flow and diffusion models, multimodal learning, and causal interpretability. I study how score-based and flow-matching generative models behave in structured latent spaces, asking questions like: how does energy evolve during a diffusion trajectory, when does classifier-free guidance hurt high-resolution quality, and can a causal graph over modalities make multimodal predictions explainable? My current work (Affect-Diff) couples a VAE over text/audio/video with a conditional 1D diffusion U-Net and a Gumbel-Softmax causal attention graph, targeting emotion recognition with counterfactual explainability. I have authored three preprints on arXiv covering energy dynamics in diffusion sampling (RectifiedHR), training-free local style control in SDXL (Local Prompt Adaptation), and a systematic negative-result study of DINO feature reuse for few-shot NeRF (NeRF Few-Shot).

Before Fordham, I was an Associate Data Consultant at Decision Foundry (Bangalore) and an undergraduate researcher at Jain University, where I worked on medical image segmentation and multiclass skin lesion classification. I am actively looking for research internships, gen ai/ ml research jobs and Ph.D. positions in generative modeling, multimodal AI, and representation learning.

Selected Projects

  • Affect-Diff: Multimodal Emotion Recognition with Diffusion Prior In Progress
    Couples a causal attention graph (Gumbel-Softmax, T↔A↔V) with a VAE latent bottleneck and a conditional 1D diffusion U-Net (CFG + causal conditioning) for interpretable affect recognition on CMU-MOSEI. Counterfactual hallucination provides modality-level causal attribution.
    GitHub Paper[Draft]
  • Diffusion Detective: Interpretable Stable Diffusion
    Real-time cross-attention extraction from SD 2.1 with MAE < 10⁻⁶. CLIP embedding algebra for semantic steering (94% success). GPT-4o-mini chain-of-thought narratives. FastAPI + React 18 frontend.
    GitHub
  • No-Propagation Diffusion Transformers
    Clean PyTorch reproduction of NoPropDT (Oxford, arXiv:2503.24322). Stack of local denoising blocks, each trained with per-layer MSE, zero gradient backward through the network. ~99% on MNIST, ~76% on CIFAR-10 without backpropagation.
    GitHub
  • Causal Factor Discovery in S&P 500 Returns
    Double Machine Learning (DML) pipeline for causal ATE/CATE estimation of momentum on next-day returns. Benchmarks: OLS, Random Forest, LSTM ablation (base/deep/wide). yfinance data for AAPL, MSFT, JPM, AMZN, XOM.
    GitHub
  • MAESTRO: AI Text-to-Video Lecture System
    Full-stack platform (OpenAI script synthesis + PlayHT TTS + FFmpeg rendering) that auto-generates educational video lectures with retrieval-augmented QA bots. Live at Fordham University.
    GitHub Live
  • AutomatedBI: Agentic BI Dashboard Generator
    CrewAI multi-agent system (4 specialized agents) + Gemini that transforms raw data into interactive dashboards. 80% reduction in manual reporting time. Flask + React + PostgreSQL + Docker.
    GitHub Demo
  • H-KFX: Autonomous Bug-Fixing Agent
    GPT-4o + MCP + MongoDB Atlas RAG autonomous agent that closes Jira tickets end-to-end: reads ticket, writes fix, opens PR. 60–90 second ticket-to-PR cycle. MUG Hackathon 2025.

    Hybrid Neo4j (Dijkstra routing) + MongoDB + GPT-4o-mini natural language parsing for indoor campus navigation. React + Cytoscape.js interactive graph map.
